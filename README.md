# meme-error

A simple VS Code extension that registers a command and shows an information message.

## Features

- Adds the command `meme-error.helloWorld`
- Command title in Command Palette: `Hello World`
- Shows: `Hello World from meme-error!`

## Requirements

No additional requirements.

## Extension Settings

This extension does not currently contribute any custom settings.
